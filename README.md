# sp-homework

## Описание:

Проект sp-homework - это блоки кода из домашнего задания. Соджержит генераторы, которые позволяют финансовым аналитикам быстро и удобно находить нужную информацию о транзакциях и проводить анализ данных

### Описание файла generators.py

Данный файл содержит три полезные функции для работы с финансовыми транзакциями и генераторами:

Функции:

1. `filter_by_currency(transactions, currency_code)`
- Назначение: фильтрует транзакции по указанному коду валюты.
- Входные аргументы:
    * transactions: список транзакций, представленных в виде словарей.
    * currency_code: код интересующей валюты (например, "USD", "EUR", "RUB").
- Выход: итератор, выдающий только те транзакции, которые соответствуют указанной валюте.
2. `transaction_descriptions(transactions)`
- Назначение: извлекает описания всех транзакций из предоставленного списка.
- Входные аргументы:
    * transactions: список транзакций, представленных в виде словарей.
- Выход: итератор, выдающий описание каждой транзакции.
3. `card_number_generator(start_position, end_position)`
- Назначение: генерирует номера банковских карт в заданном диапазоне.
- Входные аргументы:
    * start_position: начальное значение диапазона.
    * end_position: конечное значение диапазона.
- Выход: итератор, выдающий номера карт в формате XXXX XXXX XXXX XXXX.


Пример использования

```python
# Фильтруем транзакции по валюте USD
filtered_transactions = list(filter_by_currency(transactions, "USD"))

# Получаем описания транзакций
descriptions = list(transaction_descriptions(filtered_transactions))

# Генерация карточных номеров от 1 до 5
generated_cards = list(card_number_generator(1, 5))

# Вывод результатов
for desc in descriptions:
    print(desc)

for card in generated_cards:
    print(card)
```

## Тестирование:
Проект проходит автоматическое тестирование с помощью инструмента pytest
#### Инструкция по запуску тестов:

Перед запуском убедитесь, что установлен пакет pytest. Если он ещё не установлен, выполните следующую команду:
```
pip install pytest
```
Затем перейдите в корневую директорию проекта и запустите тесты следующей командой:
```
pytest
```
